## [0.1.0] – Initial Release

### Added
- Modular architecture with support for:
  - Tesseract OCR
  - EasyOCR
  - FastAPI + Streamlit + CLI interface
  - Material Design GUI (Desktop via PySide6, Mobile via Compose)
- ParsBERT integration for Persian post-correction
- Input format support: Image, PDF, Subtitle, EPUB
- Multilingual detection with langdetect
- Full cross-platform structure (Linux, macOS, Windows, Android, iOS)
