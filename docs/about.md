# About OmniOCR

OmniOCR is a modular, offline-first OCR platform designed to unify OCR workflows across languages, file types, and interfaces. Built by [GeekNeuron](https://github.com/GeekNeuron) with the goal of making OCR accessible, customizable, and intelligent.

---

## Goals
- Support real-world OCR tasks: images, scans, subtitles, and PDFs
- Combine traditional OCR engines (Tesseract) with deep learning (EasyOCR, BERT)
- Offer multiple interfaces: CLI, Web, REST API
- Prioritize speed, offline support, and language extensibility

## Tech Stack
- Python 3.9+
- Tesseract / EasyOCR
- FastAPI, Typer, Streamlit
- HuggingFace Transformers (for AI postprocessing)
- Docker, Pytest, GitHub Actions
