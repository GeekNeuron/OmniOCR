OmniOCR

Offline, multilingual OCR toolkit for image, video, and PDF recognition.
Works on Windows, Linux, macOS, Android & iOS.

Downloads

Download Latest Release

Core Features

  • Offline OCR using native Tesseract engine
  • Supports image, PDF, and video (subtitle) OCR
  • Multi-language: Persian, Turkish, Japanese, Chinese, Arabic...
  • Export to .txt, .pdf, .docx, .epub, .srt, .mobi, .azw3
  • Modern GUI (PySide6) + Mobile Compose UI

Platforms

  • Desktop: Windows, macOS, Linux (AppImage/.exe)
  • Mobile: Android (APK) & iOS (TestFlight)
