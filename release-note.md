# OmniOCR — Release Notes (v1.0.0)

## Highlights
- Multilingual offline OCR (Persian, English, Arabic, Chinese, etc.)
- Subtitle image OCR (sub/idx to SRT)
- FastAPI REST interface
- Streamlit-based web interface
- Text cleanup and normalization module
- Docker + docker-compose deployment ready
- Unit testing and modular codebase

## Tested on
- Ubuntu 20.04, WSL2
- Windows 10/11
- Python 3.10+
